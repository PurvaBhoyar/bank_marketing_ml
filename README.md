# Bank Marketing Subscription Prediction

Machine learning project to predict bank customer subscription likelihood using 8 classification algorithms: Logistic Regression, SVM, KNN, Decision Tree, Random Forest, Gradient Boosting, XGBoost, and LightGBM. Dataset: UCI Bank Marketing (41,188 records, 20 features). Best Model: Gradient Boosting (92.41% accuracy, 0.6308 F1-Score). 

Includes Streamlit web app for real-time predictions.

## Quick Start
pip install -r requirements.txt

streamlit run app.py
